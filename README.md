# Java Console Calculator

## Description
This project is a simple console-based calculator built using Java. It performs basic arithmetic operations and demonstrates fundamental programming concepts such as methods, loops, and user input handling.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Continuous operation until user exits

## Technologies Used
- Java
- IntelliJ IDEA

## Project Structure
Calculator/
└── src/
    └── Calculator.java

## How to Run

### Using IntelliJ IDEA
1. Open the project in IntelliJ
2. Run the Calculator.java file

### Using Command Line
```bash
javac Calculator.java
java Calculator
